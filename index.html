Indledning til JavaScript

1. Indledning

Javascript er browserens sprog, det er her man kan manipulere og arbejde med HTML og CSS.

Javascript udføres heller ikke bare en gang, når websiden indlæses, ligesom i HTML og CSS.

Javascript kan blive udført mange gange i løbet af den tid websiden vises i browseren.

Javascript foregår på klientsiden, hvilket vil sige at den Javascript, der er blevet tilføjet på en webside, hentes ned sammen med HTML'en og CSS'en, når en webside vises i en klients browser og så udføres koden først herefter på klientens maskine i browseren. Man har derfor altid adgang til al den Javascript kode, som en webside indeholder.

En god ting ved Javascript er, at man med Javascript instruktionerne har adgang til og kan ændre på HTML'en og CSS'en, mens websiden vises i browseren. Det kunne eksempelvis være at websiden skal skifte tekstfarven på et P tag, når brugeren klikker på P taget, hvilket kræver en ændring i CSS'en for P tagget eller at websiden skal tilføje tekst til et P tag, når brugeren klikker på en knap, hvilket kræver en ændring i HTML'en for P tagget.

Selvom man kan få Javascript til at gøre alle de samme ting, som HTML'en og CSS'en gør, skal man dog ikke fuldstændigt droppe HTML'en og CSS'en, da Javascript kun skal bruges til de opgaver, som HTML'en og CSS'en ikke kan klare i sig selv. Man skal altså bruge HTML, CSS og Javascript i en kombination, hvor de hver især, håndterer det som de er beregnet til at håndtere og på den måde adskille indhold/struktur, udseende og opførsel fra hinanden.

2. Indsættelse af Javascript

Javascript kan nogenlunde ligesom CSS tilføjes en webside på tre forskellige måder, dog lidt mere løst og der er selvfølgelig fordele og ulemper ved hver af disse tre måder, såsom overskuelighed og effektivitet. Den mest overskuelige måde man tilføjer Javascript til en webside er ved at oprette en Javascript fil, som bare er en tekstfil med endelsen .js og bagefter linke til den i HTML filen, ligesom man gør med en CSS fil. Det er dog ikke helt det samme tag man skal benytte sig af som man gør, når man tilføjer en CSS fil. Man tilføjer en Javascript fil til en webside, ved at indsætte et script tag i HTML'en med navnet på Javascript filen i src attributten.

nr. 1

Derefter kan man i filen Javascript.js tilføje de Javascript instruktioner man vil have udført. Filen behøver ikke nødvendigvis at hedde Javascript.js, men den skal bare ende på js og ellers følge de normale regler for navngivning af filer. Man kan tilføje ligeså mange Javascript filer man vil til en webside. 

<script src="Javascript.js" defer></script>

nr. 2

Hvis man derimod ikke vil have Javascript instruktionerne stående i en seperat fil, kan man undlade src attributten på script tagget og så bare skrive Javascript instruktionerne direkte inde i script tagget. 

<script>
	// javascript her...
</script>

nr. 3

Den sidste mulighed der er for tilføjelse af Javascript på en webside, er at indsætte Javascript instruktionerne direkte på hændelsesattributterne. Man kan tilføje tagsene, såsom onblur, onclick eller onload. 

<a onclick="this.style.background='red';">Skift</a>

Selvom man ikke behøver at placere scripttags i head eller body tagget i sin HTML og man i princippet kan placere dem hvor som helst i sin HTML, anbefales det dog for overskuelighedens skyld, at man adskiller Javascript koden fra HTML koden og laver seperate Javascript filer, til de Javascript instruktioner, som man ønsker at der udføres på websiden og linker til sine Javascript filer sidst i sin HTML inden body tagget lukker, da man så får læst alt sin HTML ind i browseren inden Javascripten bliver læst og tilføjet af browseren.
