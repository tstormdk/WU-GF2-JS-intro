Centrale begreber

1. Syntaks

Når man koder Javascript, er det vigtigt at overholde syntaksen.

Man skal ligesom med HTML og CSS, skrive sine instruktioner på en bestemt måde for at browserne kan forstå hvad der skal udføres og bare et enkelt forkert tegn kan gøre at alle de instruktioner man har skrevet ikke udføres ordentligt.

Det er derfor som med alle andre sprog, en vigtig del af det at lære at kode i Javascript, at lære hvordan man opbygger en instruktion.

Det kan føles som om at man skal kunne hele sproget før man kan lave en lille del. Efterhånden som man får arbejdet med JavaScript, får man en større forståelse for hvordan det virker.

2. Variabler

En variabel bruges i Javascript til at holde styr på information. Man opretter altså en variabel i sit Javascript program hvis man vil gemme noget, såsom et navn, et tal eller et resultat.

Den variabel man opretter giver man et navn og når man så vil gemme information i eller læse information frn variabeln bruger man navnet i Javascript koden.

En variabel kan se således ud.

 let fornavn = "Søren";

Hvis vi lige ser på hver enkel del af denne linje

Definerer at dette er en variabel.
fornavn
Navnet pn variabeln. Det er op til dig at give det et navn af betydning.
=
Her siger du an variabeln skal indholde en værdi.
"Søren"
Dette er værdien der skal tildeles, dette er også en type af string.
;
Dette betyder at linjen slutter her eller at der kommer ny kode efter semikolon.

En variabel kan også være af forskellige typer

String
Dette kan være en kombination af bogstaver og tal. 
let fornavn = "Søren";

Boolean
Dette kan enten være true eller false. Bruges for det meste når der skal spørges om noget. 
Let noget = true;

Number
Dennne indholder kun tal, det kan være 5 eller 5.5. 
let antal = 5;

undefined/null
Denne har ingen værdi. 
let intet;
let intet = null;

Array
Denne kan indholde mange værdier og af flere typer. Denne type kan blive meget mere kompleks. 
let bruger = new Array["Søren","Sven","Tim"];

Object
Et Object kan ses som en Array med værdier og metoder, Alt er et Object pånær undefined og null.

3. Operatorer

Operatorer er et eller flere tegn såsom +, -, =, +=, ++, == for bare at nævne et par stykker af de mest brugte. Man bruger operatorer i mange af de instruktioner man laver i et Javascript program.

Operatorer bliver i Javascript brugt når man vil gøre noget, eksempelvis lave udregninger, ændre værdi eller spørge om to værdier er ens.

Et meget lille eksempel er vist her.

let navn = "tim"let tal = 5;
                    let besked = navn + " er " + tal + " år."; // vil retunere: tim er 5 årlet erDetSandt = tal == 5; // vil retunere: true

For at få den fulde forståelse skal der arbejdes med if og if else og du vil komme til at møde dette mange gange fremover.

4. Valg - betingede sætninger

I Javascript programmer er der tit brug for at man er afhængig af om noget er sandt eller falskt. For eksempel hvis man trykker på en knap i en formular, og hvis man ikke har skrevet et navn i et input felt, skal der vises en besked til brugeren om at input feltet mangler en værdi.

Til at lave valg i et Javascript program bruger man såkaldte if eller if else strukturer omkring sine instruktioner. Hvis man bare vil have noget udført hvis noget er sandt, nøjes man med en if struktur. Hvis man også vil have noget udført når noget er falskt bruger man en if else struktur.

If

if (...) {
...kode bliver udført..
}

if else

if (...) {
...kode bliver udført..
}else{
... Hvis if ikke er sandt så udfør denne kode ...
}

Flere en et spøsmål

if (...) {
... udfør hvis dette er sandt ...
}else if(...){
... eller udfør dette hvis den forrige er falsk og denne er sand ... 
}else{
... Hvis ingen er sand så udfør denne kode ...                                            
}

Ok det giver jo ingen mening - så lad os se på noget rigtig kode

let fornavn = "Tim";
                        
if (fornavn == "Søren") {
    // dette er false
} else if (fornavn == "Tim") {
    // Dette er sandt og denne if vil blive udført.
}

Det kan tage noget tid at få en god forståelse af dette og at lære mange af de forskellige operatorer at kende.

5. Gentagelser - løkker

Vil man have noget gjort mange gange i sit Javascript program, behøver man ikke gentage sin instruktion det antal gange man vil have den udført. Man kan i stedet bruge en for, en while eller en do while struktur omkring sin instruktion.

Gentagelsesstrukturerne benytter boolske udtryk til at afgøre om instruktioner skal gentages eller om der skal fortsættes videre i programmet.

Vi kommer også til at bruge document.write(); som kan skrive ud til html siden.

                        //For
for let i = 0; i < 5; i++ )
{
    document.write( i );
}

//while
let i = 0;
while ( i < 5 )
{
    document.write( i );
    i++;
}
                          
//do while
let x = 0;                      
do{
    document.write( x );
    x++;
}while(x<5);

Du kan prøve det af i en CodePen

Du kan også se test af dette i en CodePin - husk at bruge Chrome eller Firefox ctrl + Shift + J for at se.

6. Funktioner

Man kan selv lave sine egne funktioner i Javascript. Fordelen er at man kan samle en masse kode og give koden et funktionsnavn. Man kan derefter bare skrive navnet på funktionen og så får man udført al den kode man har samlet i funktionen. 

skrivTilHtml();

function skrivTilHtml() {
    document.write("Dette kommer fra en function");
}

Funktioner kan også modtage input, for eksempel en alder som man sender ind i sin funktion, hvor funktionen så gør noget med den alder den får som input. Input til funktioner kalder man for parameter og en funktion kan sagtens modtage flere parametre eller sende noget tilbage igen som output. 

skrivTilHtml("Dette kommer fra en parameter");

function skrivTilHtml(msg) {
    document.write(msg);
}
